  #### Описание:

    Приложение для поиска вакансий, которое использует API HeadHunter. С его помощью можно легко найти подходящие вакансии, отфильтровав их по региону, зарплате и отрасли. Также есть возможность добавлять понравившиеся вакансии в список избранных.
